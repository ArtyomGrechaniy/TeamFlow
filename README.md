# TeamFlow

**Управление задачами и расходами внутри команд**

TeamFlow — это удобное веб-приложение, которое помогает небольшим командам (фриланс, стартапы, учебные группы) вести совместный учёт расходов и задач в одном месте.

---

## Основные возможности

- **Команды и роли** — владелец, администратор, участник
- **Присоединение по коду** — приглашение в команду через уникальный 12-символьный код
- **Расходы** — личные и командные, поддержка нескольких валют (RUB, USD, EUR, CNY) с автоматическим пересчётом в рубли
- **Задачи** — создание, назначение исполнителя, статусы, приоритеты и дедлайны
- **Категории** — отдельно для расходов и задач (личные и командные)
- **Статистика** — общая аналитика расходов и задач
- **Разделение контента** — чёткое разделение личного и командного пространства

---

## Технологии

- **Backend**: Python + Django 5
- **База данных**: PostgreSQL
- **Frontend**: Bootstrap 5 + custom CSS
- **Аутентификация**: Django auth + custom Profile
- **Дополнительно**: custom mixins для контроля доступа, FormContextMixin, аннотации, оптимизированные запросы
